Networking / wifi

List network devices and some details about their status
$ ip link
To connect to a new wifi network
# wpa_passphrase ssid passphrase
Copy that info to a .config file
Connect to that network
# wpa_supplicatn -B -i interface -c config file
start dhcpcd
# dhcpcd interface
If the wifi interface wont go up
$ rfkill {list|block|unblock}

Copy kernel files to EFI bootable partition: aries

Boot to a UEFI bootable usb stick
Double check which dev and partition your root and EFI are on:
# lsblk -f
Make a dir to mount the efi dir
# mkdir -p /mnt/boot
Mount the root fs and the efi partition
# mount -t ext4 /dev/sda2 /mnt
# mount -t vfat /dev/sda1 /mnt/boot
Change root
# arch-chroot /mnt /bin/bash (If sh shell is OK for you, you can omit "/bin/bash")
Copy the new kernel fils to EFI partition
# cd /boot
# cp vmlinuz-linux ./EFI/arch/vmlinuz-arch.efi
# cp initramfs-linux.img ./EFI/arch/initramfs-arch.img
# cp initramfs-linux-fallback.img ./EFI/arch/initramfs-arch-fallback.img
Update UEFI boot loader
# sudo efibootmgr -c -w -d /dev/sda -p 1 -l '\EFI\refind\refind_x64.efi' -L "rEFInd"
exit arch-chroot
# exit
Unmount
# cd /
# umount /mnt/boot
# umount /mnt
Reboot
Remove USB stick
# systemctl reboot
